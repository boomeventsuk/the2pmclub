About THE 2PM CLUB

THE 2PM CLUB is the Midlands' original daytime disco, run by Boombastic Events. We host 80s, 90s and 00s afternoon events across six cities: Northampton, Bedford, Milton Keynes, Coventry, Luton and Leicester. Events run Saturday afternoons, 2pm to 6pm.

Data Feeds

Events JSON

/events.json — All upcoming events with dates, venues, pricing, booking URLs. Updated at build time and after Eventbrite syncs.

Schema: array of event objects with fields: id, slug, title, location, start (ISO 8601), end, bookUrl, description, fullDescription, price, priceCurrency, priceLabel, statusLabel, availability (schema.org URL), venueAddress (PostalAddress).
